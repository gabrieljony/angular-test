## Instruções
Jasmine é um framework de testes
Jasmine recomenda a estrutura "Should... when..." na descrição dos testes com a função it.


Karma é um executor de testes do jasmine
Karma é responsável por executar os testes e roda em background. O objetivo é desenvolver e executar em tempo real.
Karma automatiza a execução de testes escritos em Jasmine.
